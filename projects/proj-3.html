Fall 2015

Technologies: MATLAB, Python, Keras, scikit-learn
Key Words: single-cell, ADMM, robust PCA, deep autoencoder, gene network

I recently had the pleasure of collaborating with Vasilis Ntranos, Govinda Kamath, Lior Pachter, and my advisor David Tse in developing a novel approach for clustering the cells in single-cell RNA-Seq datasets. Inspired by Lior's group's recently developed tool, kallisto, Vasilis, Govinda, and I generated feature vectors for cells based on transcript-compatibility counts rather than quantified gene abundances. The latter requires read alignment, a computationally costly approach especially as the number of reads scale with the increasing number of cells involved in single-cell RNA-Seq experiments. The former requires only pseudoalignment, or a very fast assignment of each read to a set of potential transcripts the read can come from using a k-mer hashing approach.

We showed that clustering on TCCs is both a fast and potentially universal approach by reanalyzing two recently published but quite different single-cell datasets. The work was submitted to Genome Biology's Single Cell Omics Special Issue, and our analyses and method are publicly available on GitHub. The preprint is also available here. Take a look!

Fall 2015

After a research collaboration with some members of Serafim Batzoglou's group, I was inspired to explore how deep networks could improve the analysis of single-cell data. This project was done for the Stanford artificial intelligence course (CS 221) taught by Percy Liang. The results of a single-cell experiment can be summarized using a design matrix where each entry represents the expression level of a certain gene in a certain cell. The purpose of the project was to leverage the power of deep neural networks to clean the noisy design matrix such that biologically meaningful clusters could be uncovered using simple unsupervised learning algorithms such as k-means clustering.

The most challenging parts of the project involved solving a non-convex optimization problem using ADMM and building an appropriate network with which to process the data, neither of which I had done before. With the help of a postdoc in my group, I found a convex relaxation to the problem and derived the ADMM update equations. This was also my first practical experience working with deep networks, and luckily the Keras documentation was fantastic.
